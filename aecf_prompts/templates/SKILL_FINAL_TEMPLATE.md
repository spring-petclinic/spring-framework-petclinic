# Skill Final Output Template

Use this template when a read-only, documentation, audit, or intelligence skill reaches its governed terminal phase and no skill-specific template exists.

## Metadata

| Field | Value |
|---|---|
| Document Type | SKILL_FINAL_TEMPLATE |
| Phase | TEMPLATE |
| Version | v1 |
| Status | baseline |
| Scope | prompt-only |
| Owner | SEACHAD |
| Created | YYYY-MM-DD |
| Last Updated | YYYY-MM-DD |

## 1. Objective

- Restate the governed goal of the skill terminal phase.

## 2. Scope And Evidence

- Identify the effective scope.
- Summarize the exact upstream artifacts or repository evidence used.

## 3. Analysis Or Synthesis

- Present the core findings, explanation, strategy, or generated structured result.

## 4. Risks And Constraints

- State material assumptions, limitations, unresolved questions, or compliance caveats.

## 5. Recommended Next Steps

- List concrete follow-up actions or next AECF skills when relevant.

## 6. EXECUTIVE_REVIEW

- Intended outcome: one short sentence stating what this skill execution was trying to achieve.
- Phase summary: one bullet per executed phase with a one-line result for each phase.
- Final result: one short paragraph with the final conclusion, delivered artifact, or final verdict.
- Test summary: if code changed, summarize what tests were added or run and the result; otherwise state `No code changes, so no test execution was required.`
- Security review summary: if code changed, summarize the security audit/review performed and its result or residual risk; otherwise state `No code changes, so no additional security audit was required.`
