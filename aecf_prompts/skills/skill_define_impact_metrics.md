## Skill ID
`aecf_define_impact_metrics`

## Description
Defines technical and operational impact metrics with baseline, target, measurement method and success criteria for monitoring changes.

## Objective
Establish an auditable measurement framework that connects technical decisions with business results and governance readiness.

## Inputs

### Mandatory:
- **Scope**: feature, module or initiative to measure
- **TOPIC** (optional): report identifier

### Optional:
- Historical metrics available
- Business/operation objectives
- Instrumentation restrictions
- Existing SLO/SLA

## Steps

### Step 1: Outcome Definition
Define expected outcomes (business, operation, technical quality).

### Step 2: Metric Selection
Select leading/lagging indicators and map them to outcomes.

### Step 3: Baseline Establishment
Document current baseline, data sources and observation windows.

### Step 4: Target & Threshold Definition
Define post-change goals, alert thresholds and success criteria.

### Step 5: Measurement Governance
Assign owner, measurement frequency, calculation method and traceability.

### Step 6: Reporting Specification
Generate final metric specification ready for execution.
