## EXECUTION MANDATE (IMPERATIVE)

When this skill is invoked, the AI MUST:

1. **REQUIRE TOPIC** — This skill CANNOT execute without an explicit TOPIC
2. **SCAN** `aecf_prompts/<DOCS_ROOT>/<user_id>/<RUN_DATE>/{{TOPIC}}/` for all AECF-generated documents
3. **ANALYZE** all documents found (changes, impacts, metrics, verdicts)
4. **CONSOLIDATE** incremental + global executive state
5. **CREATE FILE** at `aecf_prompts/<DOCS_ROOT>/<user_id>/<RUN_DATE>/{{TOPIC}}/AECF_<NN>_EXECUTIVE_SUMMARY.md`
6. **FOLLOW** `templates/EXECUTIVE_SUMMARY_TEMPLATE.md` exactly
7. **APPEND CHANGELOG ENTRY** at `aecf_prompts/<DOCS_ROOT>/<user_id>/AECF_CHANGELOG.md` in the target project workspace

**MANDATORY POST-EXECUTION GOVERNANCE (per SKILL_DISPATCHER)**:
- **UPDATE** `aecf_prompts/<DOCS_ROOT>/<user_id>/AECF_TOPICS_INVENTORY.json` for TOPIC lifecycle and **REGENERATE** `aecf_prompts/<DOCS_ROOT>/<user_id>/AECF_TOPICS_INVENTORY.md` (Step 4.1)
- **APPEND** one execution entry to `aecf_prompts/<DOCS_ROOT>/<user_id>/AECF_CHANGELOG.md` (Step 4.2)

**FORBIDDEN**:
- ❌ Executing without TOPIC
- ❌ Responding only in chat without creating a file
- ❌ Deviating from the EXECUTIVE_SUMMARY template structure
