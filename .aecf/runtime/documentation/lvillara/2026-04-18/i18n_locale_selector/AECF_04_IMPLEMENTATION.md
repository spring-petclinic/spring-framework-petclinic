## Files Modified

### 1. [mvc-core-config.xml](src/main/resources/spring/mvc-core-config.xml)

**Added** (after `<mvc:annotation-driven>`):

- `CookieLocaleResolver` bean (`id="localeResolver"`, `defaultLocale=en`, `cookieName=PETCLINIC_LOCALE`, `cookieMaxAge=-1`)
- `LocaleChangeInterceptor` bean (`id="localeChangeInterceptor"`, `paramName=lang`)
- `<mvc:interceptors><ref bean="localeChangeInterceptor"/></mvc:interceptors>`

**Rationale**: Spring MVC looks up the `LocaleResolver` by the bean name `localeResolver` (fixed convention). Without this bean, it defaults to `AcceptHeaderLocaleResolver` which cannot be changed programmatically.
